3. The Agent Architecture

Research Agent

Scanned docs, pricing pages, and API references for all 100 apps to determine auth type, gate status, and API breadth without human input.

Execution Agent

LangGraph ReAct agent with Composio tool-routing. Powered by NVIDIA LLMs. Attempts live auth checks and fetches tools autonomously.

Human-in-the-Loop

OAuth consent gates require a human to visit the connect.composio.dev link. Composio's managed-auth flow makes this the only friction point.

5. Verification (Accuracy Check)

Hits (Successes)

  • Successfully connected 84 out of 100 tools instantly via Composio catalog.
  • LangGraph autonomously fetched the tool catalog schema for every connected app without hardcoded endpoints.
  • Auth flow correctly paused for human-in-the-loop OAuth consent when required.

Misses (Honest Failures)

  • Validation Errors: The agent originally hallucinated specific IDs (e.g., `profile_id` for Slack) causing execution failure. We had to constrain the prompt to only use zero-arg read tools.
  • Enterprise Gates: 16 tools (like Bloomberg or legacy ERPs) simply refused connection without a commercial contract. Autonomous agents cannot bypass this.
